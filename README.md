# Frontend Registration Form (Beginner Level)

This project is a **beginner-level HTML task** created to practice basic frontend development concepts. It focuses on building a simple and structured registration form using only HTML.

## 📌 Project Description
This project demonstrates how to create a clean and user-friendly registration form using fundamental HTML tags. It is designed as a learning task for beginners who are starting their journey in frontend development.

The webpage includes a form to collect user details, along with embedded media and external links to enhance the overall layout.

## ✨ Features
- Proper HTML page structure
- Registration form using table layout
- Input fields for name, age, email
- Gender selection using radio buttons
- Course selection using dropdown menu
- File upload option for resume
- Embedded YouTube video
- External navigation links

## 🛠️ Technologies Used
- HTML5

## 🎯 Learning Objectives
This beginner task helps in understanding:
- HTML document structure
- Form elements and input types
- Table-based layouts
- Embedding external content using iframe
